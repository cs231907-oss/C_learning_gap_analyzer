# C Learning Gap Analyzer

A rule-based, AI-assisted learning gap analyzer designed to identify conceptual and practical mistakes in C programming.  
The system evaluates both **theory answers** and **practical C programs**, providing **explainable, student-friendly feedback**.

---

## Features
- Practical C code analysis (printf / scanf / headers)
- Theory question evaluation using keyword-based logic
- Error categorization with severity levels
- Explainable feedback (no black-box AI)
- Streamlit-based interactive interface

---

## Technologies Used
- Python
- Streamlit
- Rule-based semantic analysis
